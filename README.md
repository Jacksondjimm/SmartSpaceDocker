# SmartSpaceDocker
Программное обеспечение для сервера домашнего климат контроля (на базе Raspberry Pi).

Папка bme280 содержит программный код среды Arduino IDE для платы на базе ESP-01 и датчика температуры и влажности bme280 (далее Wi-fi датчик температуры).
Wi-fi датчики температуры размещены в разных комнатах квартиры.

Папка WebApplication содержит программный код на базе C# Razor Pages для подключения к Postgre SQL server.

Всего установлено 3 Wi-fi датчика температуры из 5. Каждый Wi-fi датчик температуры транслирует данные на сервер по Wi-fi.

На сервере работает контейнер docker c проектом Razor Pages на базе ASP.NET и контейнер docker с Postgre SQL 
(развёрнуто на Raspberry Pi). По каждому датчику глубина архива составляет 1000 показаний в данной версии программы. Архив пишется "по-кругу".
Данные от датчиков передаются 1 раз в 5 минут. В планах добавить пагинацию и усложнить код для автоматического подключения новых датчиков.
